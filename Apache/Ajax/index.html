Ajax



Es una vieja tecnología que fue re descubierta para crear más fácilmente aplicaciones web, funciona de la siguiente manera, lo que normalmente se hace en una página web, se piden datos al servidor, en internet se ubica el servidor, el navegar despliega la vista de la página web, anteriormente el navegador con la url la mandaba al servidor, el servidor respondía con el contenido, para mandar datos de un formulario de la página simplemente se recargaba para mostrar el nuevo contenido, pero los chats y nitificaciones de face, el mapa de googlemaps son medios que trabajan en tiempo real, y funcionan gracias a la tecnología Ajax.

Ajax es un pedazo de tecnología ubicado en el navegar, Ajax significa Asyncronous JavaScript And XML, aunque el xml ya no se usa, se mantiene solo por el nombre, a través desde el navegador usando javascript se le dice al navegador traer más datos, sin necesidad de recargar, ya que trabaja por áreas, el concepto más importante es asíncrono, que significa que no hay necesidad de recargar la página, otro proceso es json, una forma mucho mas compacta de transmitir datos. Se tiene un servidor web, se tiene también una base de datos, conectados a un sitio web, con estructuras provenientes de la base de datos, normalmente se necesitaría recargar la página para recargar los datos, a través de Ajax, a través de un lenguaje de programación json crea un formato e interactúa con Ajax quien estructura al navegador y muestra la página web con datos provenientes de servidor.
